Var, Let, Const

VAR se la puede utilzar en varias partes y ademas acepta cualquier tipo de caracter: Strings, Int, Object, Array, Function. Es usado en anteriores versiones de JavaScript. Se puede acceder a un VAR desde cualquier parte de una funcion.


var a = 12;

Una variable puede ser definida con una cadena, más tarde, reasignada a un entero y más tarde podría ser una función. Eso es un poder pero también una debilidad


  var   variableGlobal ; 
  variableGlobal   =   'Amo la pizza!' ; 
  console . log ( variableGlobal ) ;

LET let permite asignar valor a nuestras variables con visibilidad / validez únicamente dentro del contexto léxico (llaves) en el que se ha definido:


    if ( true ) {
    let y = 'Hola Mundo';
    console.info( y ); // 'Hola Mundo'
    }
    console.info( y ); // ReferenceError: y is not defined

CONST es como let excepto que no se pueden asignar, excepto en el punto donde son declaradas.


( function ( )   { 
     const   constVar1   =   'const variable 1 value in parent scope' ; 
 
     ( function ( )   { 
         // Scope 1 
         const   constVar1   =   'const variable 1 value in scope 1' ; 
         console . log ( 'scope 1: ' ,   constVar1 ) ; 
         // scope 1:  const variable 1 value in parent scope 
     } ( ) ) ; 
 
     console . log ( 'Patent scope: ' ,   constVar1 ) ; 
     // Patent scope:  const variable 1 value in parent scope 
 
 } ( ) ) ;
