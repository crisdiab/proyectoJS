ARROW FUNCTIONS

Conocidas como funciones flecha o de dirección, son funciones definidas con una nueva sintaxis que utiliza una "flecha" ( => ) como parte de la sintaxis. Sin embargo, las funciones arrow, comportan de manera diferente a las funciones JavaScript tradicionales

¿Cuántas veces has programado un código con una estructura similar a la siguiente?


 // ES5
// Imaginemos una variable data que incluye un array de objectos
var data = [{...}, {...}, {...}, ...];  
data.forEach(function(elem){  
    // Tratamos el elemento
    console.log(elem)
});

Con la función arrow => de ES6, el código anterior se sustituiría por:


    //ES6
var data = [{...}, {...}, {...}, ...];  
data.forEach(elem => {  
    console.log(elem);
});

var miFuncion = (num) => num + num;


var miFuncion = (num) => num + num;
