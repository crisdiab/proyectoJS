TEMPLATE LITERALS

Las plantillas de cadena de texto (template strings) son literales de texto que habilitan el uso de expresiones incrustadas. Es posible utilizar cadenas de texto de más de una línea, y funcionalidades de interpolación de cadenas de texto con ellas

STRING INTERPOLATION

Interpolación de expresión intuitiva para cadenas de una sola línea y multilínea. (Aviso: no se confunda, los literales de plantilla se denominaron originalmente "cadenas de plantilla" en los borradores de la especificación de lenguaje ECMAScript 6) 

var customer = { name: "Foo" }
var card = { amount: 7, product: "Bar", unitprice: 42 }
var message = `Hello ${customer.name},
want to buy ${card.amount} ${card.product} for
a total of ${card.amount * card.unitprice} bucks?`
en ES5 era 

var customer = { name: "Foo" };
var card = { amount: 7, product: "Bar", unitprice: 42 };
var message = "Hello " + customer.name + ",\n" +
"want to buy " + card.amount + " " + card.product + " for\n" +
"a total of " + (card.amount * card.unitprice) + " bucks?";
CUSTOM INTERPOLATION

Interpolación de expresión flexible para métodos arbitrarios. 

get`http://example.com/foo?bar=${bar + baz}&quux=${quux}`
en ES5 era 

get([ "http://example.com/foo?bar=", "&quux=", "" ],bar + baz, quux);
RAW STRING ACCESS

Acceda al contenido de la cadena de plantilla en bruto (las barras invertidas no se interpretan). 

function quux (strings, ...values) {
    strings[0] === "foo\n"
    strings[1] === "bar"
    strings.raw[0] === "foo\\n"
    strings.raw[1] === "bar"
    values[0] === 42
}
quux `foo\n${ 42 }bar`

String.raw `foo\n${ 42 }bar` === "foo\\n42bar"
