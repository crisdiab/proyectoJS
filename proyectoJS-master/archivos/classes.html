Classes

ES6 nos permite de una manera simplificada la creacion de clases

ejemplo: 



                   class Person {
                        constructor(nombre, edad, genero) {
                            this.nombre   = nombre;
                            this.edad    = edad;
                            this.genero = genero;
                        }
                    //Metodos,Operaciones
                        incrementarEdad() {
                          this.edad += 1;
                        }
                    }
Ademas si queremos realizar herencia necesitamos agregrar la palabra "extends" en la clase como se muestra en el siguiente ejemplo.

Como se puede observar ES6 en lo referente a clases no permite escribir codigo limpiio lo que es una buena practica para que los principiantes entiendan de una buena manera.
