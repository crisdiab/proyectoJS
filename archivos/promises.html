Promises

Nos permite convertir nuestros callbacks de codigo Horizontal en vertical


                      func1(function (value1) {
                          func2(value1, function (value2) {
                              func3(value2, function (value3) {
                                  func4(value3, function (value4) {
                                      func5(value4, function (value5) {
                                          // Do something with value 5
                                      });
                                  });
                              });
                          });
                      });

Este codigo puede ser confuso de entender para saber el valor de retorno. para esto usamos la forma vertical mostrada a continuacion


                      func1(value1)
                          .then(func2)
                          .then(func3)
                          .then(func4)
                          .then(func5, value5 => {
                              // Do something with value 5
                          });
Para Promisses solia usarse bluebird o Q, actualmente se tiene promisse nativas, las cuales tienes dos manipuladores resolve y reject. Resolve si la prommise se cumple Reject si la promise se rechaza
