  • filter_dramaDefault Parameters

    En ES5, que habíamos variando las formas de manejar funciones que necesitan valores por defecto , argumentos indefinidos , y parámetros con nombre . Con ES6, podemos lograr todo esto y mucho más utilizando una sintaxis más concisa.

    Ejemplo de codigo

    En ES5

    
                     
                     function  SumarNumeros ( x , y ) {
                        x = x ||  0 ;
                        Y = Y ||  0 ;
                        return x + y;
                    }

    En la funcion se declara valores por defecto igual a cero o el valor por argumento si la variable no es enviada en la invocacion automaticamente se asigna

    En ES6

    

                    función  SumarNumeros ( x = 0 , y = 0 ) {
                         retorno x + y;
                    }

    Es una forma mas simplificada la que se ofrece

    Resultados
    
                     
                     SumarNumeros(3,4) //Respuesta 3+4=7
                     SumarNumeros(2) //Respuesta 2+0=2
                     SumarNumeros() //Respuesta 0+0=0
  • placeRest Parameters

    Anterior en ES5 el numero indefinido de argumentos se manejaba de la siguiente manera

    
                     function ejemplo(x,y){
                     var a = Array.prototype.slice.call(arguments, 2);
                     
                     return x+y+a.length;
                     }
    En ES6: 
    
                       function ejemplo(x,y,..a){
                    
                     return x+y+a.length;
                     }

    Es necesario simplemente en nuestra funcion poner "..a" asi tendremos la opcion de escribir argumentos infinitos

    Resultados
    

                    ejemplo(2,3,4,5) //2+3+2=7
                    ejemplo(3,1,false) //3+1+1=5
  • whatshotNamed Parameters

    Uno de los patrones en ES5 para manejar "Named Parameters" era utilizar las opciones de objeto de modelo, adoptado de jQuery.

    En ES5: 
    
                     
                     function initializeCanvas(options) {
                        var height = options.height || 600;
                        var width  = options.width  || 400;
                        var lineStroke = options.lineStroke || 'black';
                    }
    En ES6: 
    
                     
                     function initializeCanvas(
                        { height=600, width=400, lineStroke='black'}) {
                            // Use variables height, width, lineStroke here
                        }
    La solucion consiste en dejar en blanco dentro de la funcion asi si queremos cambiar cualquier valor lo hacemos
